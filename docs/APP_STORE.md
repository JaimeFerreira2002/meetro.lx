# App Store publishing checklist (iOS)

Not legal advice ‚Äî a working checklist toward a BETA/TestFlight and App Store
submission. Items marked **[you]** require the account holder.

## Legal / must-resolve before public release
- [ ] **[you] Confirm Metro Lisboa's API terms** permit a published third-party app
      (redistribution of `EstadoServicoML` data). This is the biggest external risk.
- [x] **Name:** **meetro** (`.Lisboa`) ‚Äî distinct, doesn't imply an official Metro
      Lisboa product. Own logo in `assets/logos/`.
- [ ] **Trademark ‚Äî still open:** the app shows Metro de Lisboa's **logo**
      (bottom-left credit) and their **line pictograms**. Those are their marks; using
      them can read as official, which our own Terms disclaim. Keep the "unofficial"
      disclaimer prominent and get comfortable with this before a public submission
      (a common middle ground: their logo only in About next to "Data provided by",
      not persistently on the map).
- [ ] **Tile usage limits:** CARTO's free basemaps and OSM/Nominatim public servers
      are **not** licensed for high-volume production. For real traffic, move to a
      paid tile plan and a proper geocoding provider (or self-host).
- [ ] **Attribution** shown in-app (map credit + About/credits) ‚Äî done; keep it.
- [ ] Host **[PRIVACY.md](PRIVACY.md)** at a public URL; add it in App Store Connect.
- [ ] Review **[TERMS.md](TERMS.md)**; replace all `[DATE]` / `[CONTACT]` placeholders.

## Apple setup **[you]**
- [ ] Enroll in the Apple Developer Program (~$99/yr).
- [ ] Create the App Store Connect record; set bundle ID and final display name.
- [ ] Privacy "nutrition labels": declare **Precise Location** ‚Äî used for *App
      Functionality*, **not** linked to identity, **not** used for tracking. Note the
      third-party search request to Nominatim.
- [ ] Export compliance: app uses only standard HTTPS encryption ‚Üí typically exempt.
      Add `ITSAppUsesNonExemptEncryption = false` to `Info.plist` to skip the prompt.
- [ ] Age rating questionnaire (expected 4+).
- [ ] Screenshots (required sizes), description, keywords, support URL.

## Build / release
- [ ] `NSLocationWhenInUseUsageDescription` present ‚Äî done (in `ios/Runner/Info.plist`).
- [ ] Commit the generated `ios/` folder (so the plist + entitlements persist) or
      script these changes; today they're regenerated by `flutter create`.
- [ ] Archive a release build, upload to TestFlight, run an internal beta.
- [ ] Submit for review.

## Bundled legal (in-app)
- Privacy Policy and Terms of Use are viewable in **Settings ‚Üí About & credits**
  (`app/lib/legal.dart`). Keep them in sync with the hosted versions.
